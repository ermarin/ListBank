
# Challenge

## Correr el proyecto local:

```bash
# Una vez que se clono el proyecto navegar hacia el proyecto
cd ListBank/
# Usando npm
npm i
npm run start

# O utilizando Yarn
yarn i
yarn start
```

## Visualizar el proyecto en emulador

### Para Android

```bash
# Una vez que levanto el proyecto de forma local con exito se puede presionar la tecla
a
# Para Android e
i
# Para iOS

# O se pueden utilizar los comandos

# Utilizando npm
npm run android

# O utilizando Yarn
yarn android
```

### Para iOS

```bash
# Utilizando npm
npm run ios

# O Utilizando Yarn
yarn ios
```
